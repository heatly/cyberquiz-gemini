## Getting Started

1. Create `.env` and add:
   ```
   GEMINI_API_KEY=your-api-key-here
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   python app.py
   ```

4. Open `http://localhost:5000` in your browser.
